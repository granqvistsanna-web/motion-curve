## Development

### Setup

```bash
npm install
```

### Run Dev Server

```bash
npm run dev
```

The dev server runs on `https://localhost:5173`. Add the plugin in Framer by pointing to the project directory.

### Build

```bash
npm run build
```

### Package Plugin

```bash
npm run pack
```

## Tech Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS v3
- Framer Plugin SDK v3
